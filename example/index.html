Sympathy

Effortlessly synchronize internal variables and page elements.

Create a bondable variable

var x = new Sympathy.Emitter(50);

Get the value of a variable

x();
=> 50

Set the value of a variable

x(75);
x();
=> 75 => 75

Synchronize a variable to a page element

x.link(document.getElementById('xSpan'));

x = 75

x.link(document.getElementById('xRange'));
x.link(document.getElementById('xField'));
