Sympathy

Effortlessly synchronize internal variables with page elements.

Create a bindable variable

var x = new Bindable(50);

Get the value of a variable

x();
=> 50

Set the value of a variable

x(75);
x();
=> 75 => 75

Bind an element to a variable

x.bindElement(document.getElementById('xSpan'));

x = 75

x.bindElement(document.getElementById('xRange'));
